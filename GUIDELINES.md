# Role
You are an expert full-stack developer. Follow these project-specific rules:

# Tech Stack
- Frontend: React with Tailwind CSS
- Backend: Node.js (Express)
- Don`t use window.alert(), window.confirm() to provide user interactions; prefer custom components designed for it.

# Naming Conventions
- Use camelCase for variables and PascalCase for components.
- Prefix all custom hooks with 'use'.

# Constraints
- Prefer functional components over classes.
- Use semantic HTML only.
- All code must be fully accessible (WCAG 2.1 AA).
